# 法宝能力一览

本文整理当前已经注册的法宝能力及其实际视觉表现。分类仅用于阅读，不代表能力被硬性绑定到某种器形；能力的具体强度、范围、持续时间和消耗由法宝实例的 Atom 组合决定。

| 类别 | 名字 | 效果 | 视觉效果 |
| --- | --- | --- | --- |
| 剑系 | 飞剑攻击 | 持续驱动法宝追踪不同目标并反复加速穿刺；伤害按施术者法术伤害乘法宝倍率计算。 | 法宝放大至激活尺寸，剑体实际飞行转向，附带金属残影、连续拖尾、命中粒子及起止闪光。 |
| 剑系 | 分光剑阵 | 根据能力强度生成多道独立技能剑影，持续沿敌人方向横贯圆阵并以标准扫掠碰撞伤敌。 | 固定尺寸、剑尖朝下的剑影围成旋转圆阵；约半数剑影以错落节奏从随机阵位直线飞至另一侧圆周并换位入阵，飞行时带通用残影，阵心带旋转金属符纹。 |
| 剑系 | 回锋护主 | 积蓄可恢复的防御次数，受到攻击时减伤并反击攻击者。 | 从剑尖向攻击者划出反击光束，伴随目标粒子和护主脉冲。 |
| 剑系 | 破甲剑芒 | 法宝主人造成伤害时，按目标护甲施加破甲。 | 目标身上出现六边形裂甲印记和金属火花。 |
| 鼎系 | 鼎辅炼丹 | 提高炼丹进度、缩短炼制时间并提高丹药品质。 | 炼制步骤触发鼎火，出丹时出现火焰光环、闪光和扩散波纹。 |
| 鼎系 | 鼎火炼化 | 部署持续燃烧的炼化区域，周期造成火焰伤害并施加燃烧。 | 火焰法域、鼎心火焰、向上飘散的余烬以及周期爆燃。 |
| 鼎系 | 吞纳归元 | 法域持续伤害、牵引并抽取敌人灵力，将其转化为生命和部分灵力。 | 反向旋转的吞噬法阵、向内收缩的粒子和抽取脉冲。 |
| 鼎系 | 百炼炉心 | 强化炼器生产流程，提高进度、品质和产量，并缩短耗时。 | 工序中炉心持续燃烧，完成时出现火环、火星和强光脉冲。 |
| 镜系 | 洞微灵鉴 | 提高命中、暴击和伤害上限等洞察类属性。 | 镜面持续微光，法宝周围缓慢旋转反照符纹。 |
| 镜系 | 返照反术 | 降低受到的非武器伤害，并按原有元素构成反射部分伤害。 | 镜面向施术者射出反照光束，伴随镜光闪烁和脉冲。 |
| 镜系 | 破妄照影 | 展开领域，净化友军、显露隐身敌人并驱散敌方增益。 | 旋转的十边破妄符印、净化法域和扩散净化波。 |
| 镜系 | 摄魂定魄 | 抽取单个目标灵力并返还给主人，同时造成眩晕和沉默。 | 镜面与目标间出现逆流魂索，目标脚下生成旋转定魄符印。 |
| 印系 | 山岳印落 | 法宝砸落目标区域，造成范围伤害、定身或眩晕及冲击力。 | 落点地纹、土石冲击环、碎屑、震屏和落岩音效。 |
| 印系 | 禁法敕令 | 展开禁法领域，周期沉默敌人并消耗其灵力。 | 旋转八边敕令、领域边界和周期符光。 |
| 印系 | 封脉镇身 | 封锁单个目标，使其减速、虚弱并持续流失灵力。 | 法宝到目标之间的封脉锁链和目标身上的八边封印。 |
| 袍系 | 云衣护体 | 生成可再生的护盾池，优先吸收受到的伤害。 | 主人周围持续显示护体法环，受击时出现护盾闪光和粒子。 |
| 袍系 | 遁形天衣 | 进入隐匿状态，提高移动能力并获得减伤，受击或结束时解除。 | 披帛式飘带、薄雾粒子和低强度轮廓脉冲。 |
| 袍系 | 移花卸力 | 将部分受到的伤害转移给攻击者，并产生推拉效果。 | 主人和攻击者之间出现弯曲风索，触发卸力爆发。 |
| 扇系 | 罡风横扫 | 对扇形区域造成伤害并强力击退敌人。 | 分层风系扇面、定向气流、扇缘披帛拖尾和风爆粒子。 |
| 扇系 | 离火焚风 | 释放火焰扇风，造成火伤并施加燃烧。 | 火焰扇面、扇心火光、定向火星和喷发脉冲。 |
| 扇系 | 清风涤障 | 净化扇形区域内的友军，同时驱散敌人的增益。 | 净化风扇面、流动光点和扩散清风波。 |
| 葫芦系 | 吞天葫芦 | 牵引、伤害并抽取敌人灵力，将吸收的力量储存在葫芦中。 | 葫芦口吞噬漩涡、七边符纹、内吸粒子和目标牵引魂索。 |
| 葫芦系 | 灵葫储元 | 储存主人溢出的灵力，在灵力不足时自动返还。 | 法宝周围旋转储元符纹，其尺寸和亮度随储量变化。 |
| 葫芦系 | 五炁倾泻 | 消耗储存力量进行扇形元素攻击，并按材质附加燃烧、中毒、减速、眩晕或破甲。 | 元素喷流扇面、定向粒子和爆发脉冲；当前基础表现偏火系。 |
| 灵珠系 | 周天珠护 | 形成可恢复的护盾池吸收伤害。 | 根据剩余护盾显示一至三颗环绕珠影，配合周天法环和受击波纹。 |
| 灵珠系 | 五行连珠 | 持续在目标区域发动多次元素范围打击。 | 珠影环绕目标区域，五边符阵旋转，每次打击产生扩散波和元素爆发。 |
| 灵珠系 | 连珠共鸣 | 主人施放法术时，根据发射物数量恢复灵力，受冷却和恢复上限约束。 | 五颗珠影短暂环绕主人，并产生共鸣光环和粒子。 |
| 钟系 | 震魂钟鸣 | 对周围敌人造成伤害、眩晕和击退。 | 多层魂波向外扩散，伴随金属钟声、粒子爆发和震屏。 |
| 钟系 | 清音涤尘 | 净化友军负面状态并驱散敌方增益。 | 净化音波、旋转淡出的十二边符印和大范围闪光。 |
| 钟系 | 金钟护界 | 生成有持续时间的护盾池保护主人。 | 主人上方显现放大的半透明钟影，周围带护界法环，受击时扩散金色波纹。 |
| 塔系 | 宝塔镇狱 | 部署镇狱区域，周期伤害并禁锢或眩晕敌人。 | 镇狱法域、八边封禁符印、碎片粒子及落地震动。 |
| 塔系 | 九重界障 | 部署多层防御领域，提高友军护甲或赋予界障状态。 | 两层反向旋转的护界圆环和九边界障符印。 |
| 塔系 | 塔影投界 | 在目标地点投射塔影，持续造成伤害和减速。 | 放大的半透明塔影、镇压法域、旋转符纹和周期冲击波。 |
| 幡旗系 | 万魂幡阵 | 展开魂阵，周期伤害、抽取灵力并施加虚弱。 | 魂系法域、旋转魂纹、游离魂火以及幡面到目标的抽魂链。 |
| 幡旗系 | 号令旗域 | 展开号令领域，提高友军伤害能力。 | 命令法域、旋转六边号令纹和周期闪光。 |

| 通用 | 护主灵障 | 储存可恢复的护主次数，触发时按比例降低伤害。 | 待命时法宝周围有微弱护符，受击时主人身上播放护盾动画。 |
